# markdownview-- 预览示例

编辑这个文件时，右侧预览会自动刷新。

## 常用语法

- [x] 实时预览
- [x] 表格
- [ ] 后续扩展 Mermaid 和数学公式

| 功能 | 状态 |
| --- | --- |
| GitHub Markdown 方言 | 已支持 |
| 本地相对图片 | 已支持 |
| HTML 导出 | 已支持 |

> 滚动左侧编辑器，预览窗口会按行数比例跟随。

```cpp
document->setMarkdown(markdown, QTextDocument::MarkdownDialectGitHub);
```
